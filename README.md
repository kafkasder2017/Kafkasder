## 🎯 Proje Hakkında

KAFKASDER Yönetim Paneli, sivil toplum kuruluşlarının ihtiyaçlarını karşılamak üzere tasarlanmış kapsamlı bir yönetim sistemidir. Sistem, modern web teknolojileri ve yapay zeka entegrasyonları ile güçlendirilmiş, kullanıcı dostu bir arayüz sunar.

### 🎯 Hedefler

- **Merkezi Yönetim**: Tüm STK operasyonlarını tek platformda toplama
- **Verimlilik**: Otomatik süreçler ve AI destekli karar verme
- **Şeffaflık**: Detaylı raporlama ve denetim kayıtları
- **Erişilebilirlik**: Responsive tasarım ve kullanıcı dostu arayüz
- **Güvenlik**: Rol tabanlı erişim kontrolü ve veri koruması

### 🔐 Güvenlik Katmanları

CREATE POLICY "Users can view own data" ON kisiler
  FOR SELECT USING (auth.uid() = kullanici_id);

## 🔐 Güvenlik

### 🛡️ Güvenlik Katmanları

1. **Authentication**: Supabase Auth ile güvenli kimlik doğrulama
